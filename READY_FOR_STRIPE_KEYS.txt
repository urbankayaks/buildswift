WHAT YOU NEED TO DO (5 minutes total)
================================================================================

1. Create Stripe Account (if needed):
   → Go to dashboard.stripe.com
   → Sign in or create account
   → Verify email

2. Get API Keys:
   → Go to Settings → API Keys
   → Copy PUBLISHABLE KEY (pk_...)
   → Copy SECRET KEY (sk_...)
   → Note: Keep secret key safe, don't share

3. Get Webhook Secret:
   Two options:
   
   Option A (Local Testing):
   → Run: stripe listen --forward-to localhost:5000/webhook
   → Copy the whsec_... from output
   
   Option B (Production):
   → Go to Settings → Webhooks
   → Add endpoint: https://yourdomain.com/webhook
   → Copy the signing secret

4. Update Configuration:
   → cd ~/.openclaw/workspace-silas/buildswift
   → Create .env file with your keys:
   
   STRIPE_PUBLISHABLE_KEY=pk_live_YOUR_KEY_HERE
   STRIPE_SECRET_KEY=sk_live_YOUR_KEY_HERE  
   STRIPE_WEBHOOK_SECRET=whsec_YOUR_KEY_HERE
   
   Or run: python stripe_setup.py --setup
   And follow the prompts
